API Endpoints

  • GET /api/users - Get all users
  • GET /api/users/<id> - Get a specific user
  • POST /api/users - Create a new user (requires JSON with name and email)
  • PUT /api/users/<id> - Update a user (requires JSON with name and email)
  • DELETE /api/users/<id> - Delete a user
